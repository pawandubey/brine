** Up and Running

Requires that rustc and cargo are installed and available on the PATH.

#+begin_src
  git clone https://github.com/pawandubey/brine
  cd brine

  # Build
  cargo build

  # Run repl
  cargo run

  # Run from source file
  cargo run -- /path/to/file
#+end_src
